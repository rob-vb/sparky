# Sparky MVP UI Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build the complete MVP UI for Sparky (daily validated startup ideas) with mock data, using shadcn components and the brand's esports aesthetic.

**Architecture:** Dark-mode-first design with electric blue primary (#3B82F6) and yellow/gold accent (#F59E0B) colors derived from the mascot. All pages use Next.js App Router with server components. Mock data simulates backend responses.

**Tech Stack:** Next.js 16, Tailwind CSS 4, shadcn/ui (new-york), Lucide React, Geist fonts

**Files:**
